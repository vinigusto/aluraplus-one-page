# Projeto - Alura+

Pequena landing page demonstrando a oferta "Combo+" (Alura+ + Alura Língua). Projeto de estudo com HTML e CSS.

## O que tem aqui

- `index.html` — página principal do site (landing page).
- `styles.css` — estilos do site.
- `assets/` — imagens e recursos usados pelo layout (logos, telas, background etc.).

## Como visualizar

A forma mais simples é abrir o arquivo `index.html` no navegador.

Ou abra o projeto no VS Code e use a extensão "Live Server" para servir localmente.
